# MiniADAS - Hệ thống phát hiện và cảnh báo tiền va chạm
Project môn học hệ thống nhúng - UET
Thành viên:
- Bùi Hoài Nam (23021868)
- Hoàng Bùi Thành Long (23021858)

## Mô tả hệ thống
Sử dụng STM32F401RE điều khiển:
- Cảm biến HC-SR05 đo khoảng cách.
- LCD 1602 I2C hiển thị thông số.
- Motor Driver L298N (phanh mềm tuyến tính).
- Buzzer & LED cảnh báo theo vùng nguy hiểm.

## Công nghệ sử dụng
- IDE: STM32CubeIDE
- Peripheral: I2C, Timer Input Capture, GPIO, PWM
